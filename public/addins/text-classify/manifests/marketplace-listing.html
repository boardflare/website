LIMITATIONS

  • Only suggests one tag, the one with the highest probability.
  • English-only text support.
  • Always returns a suggested tag, even if it is irrelevant.

APPLICATIONS

  • Classify customer feedback into categories.
  • Identify which social media comments contain a topic.
  • Determine frequency of help desk inquiries for by topic.
  • Sort product descriptions by topic.

HOW TO USE

  • Select a range of cells for the Text you want to classify and click Get Range.
  • Repeat the process for the Tags used for classification.
  • Click Analyze, and the results will be displayed in a new worksheet.
  • See Text Classification for Excel documentation for further details.
